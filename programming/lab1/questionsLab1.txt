Особенности Java:
Код сначала преобразуется в байт-код, независимый от платформы. Байт-код выполняется JVM (Java Virtual Machine). Не является чисто компилируемым языком. Является объектно-ориентированным языком



JDK (Java Development Kit) - в байт код
JRE (Java Runtime Enviroment) - из байт кода
JVM - интерпретирует байт код



Примитивные типы данных в Java:
byte, short, char, int, long, float, double, boolean



Декларация, инициализация переменных и присваивание значения:
[тип переменной] [имя переменной] = [значение переменной];



Ветвления:
if и switch 

"if - else":
if [условие]{
	[код];
}else if [условие]{
	[код];
}else{
	[код];
}

"switch":
swith ([выражение для выбора]){
	case (значение1):
		[код];
		break;
	case (значение2):
		[код];
		break;
	и т.д.
}



Циклы:
for ([инициализация счетчика]; [условие]; [изменение счетчика]){
	[код];
}

Форматированный вывод числовых данных:
System.out.printf("[необходимый формат вывода, где вместо % подставится х]", х);
Например:
System.out.printf("|| %8.3 ||", x);
Вывод:
|| [значение х с 8 знаками до запятой и 3 после] ||
